## Getting Started

1. Clone the Repository:
```bash
git clone https://github.com/khaymanii/netflix-UI.git
```
2. Install Dependencies:
```bash
npm install
```
3. Run the Application:
```bash
npm start
```
The dashboard will be accessible at http://localhost:3000 by default.
